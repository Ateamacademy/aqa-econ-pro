How to use these papers

  1. Click any Open Paper button to view the paper in your browser.
  2. Print it on paper, or save it as a PDF, using the Print / Save as PDF button at the top of each paper (or press Ctrl+P / ⌘+P).
  3. In the print dialog, set Margins to None or Default, turn Background graphics on, and set paper size to A4.
  4. Sit each paper under exam conditions: 2 hours, no notes, calculator permitted.
For best results, use Chrome, Edge, Firefox or Safari. The hatched "DO NOT WRITE IN THIS AREA" margin pattern relies on CSS gradient backgrounds — make sure Background graphics is enabled when you print.
